# SAM
Student Attendance Management System to help teachers manage their students's attendance.

Technology Used:

- DotNet Core 8.0  link: https://dotnet.microsoft.com/en-us/download
- C#
- JavaScript
- JQuery
- Bootstrap 5
- Ms Visual Studio
